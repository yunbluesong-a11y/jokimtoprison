### 3. 6항 핵심 논증 구조

#### 6-가: "별산제 자율" 진술 반박
→ 관리계좌 인출 27건은 피의자 본인이 실행한 거래 + 답변서(증거5) 자백과 모순

#### 6-나: "회계법인이 작성" 진술 반박
→ 거래 지시·자금 이동은 피의자가 실행 + 조서에서 가수금/가지급금 정확히 구분 → "몰랐다" 불가

#### 6-다: "송명욱 7천만원 가지급금" 진술 반박
→ 같은 적요가 정반대 방향 자금이동에 사용 + 을39 변작(141M 누락) + 계정과목 매년 변경 자인

#### 6-라: "155M은 세무서가 판단" 진술 **탄핵** ← 핵심
→ **과소신고 확인서(증거13)**: 조대제·김상균이 직접 서명하여 141,076,443원 과소신고 확인
→ 141,076,443 × 1.1 = 155,184,087원 = "세무서가 판단"했다는 바로 그 금액
→ **141,076,443원 ≈ 141,405,860원(횡령액)** — 오차율 0.23%
→ 횡령→장부조작→매출제외→과소신고 확인서 서명→책임전가(무고·소송사기) = **완결된 범죄 구조**

#### 6-마: "22504 = 송명욱 전용 계좌" 진술 반박
→ 오히려 3단계 자금세탁 구조를 피의자 스스로 자백

#### 6-바: "법원 기망 의도 없다" 진술 반박
→ 을39 변작, 갑21~24 허위서증, 가지급금 조작, "몰랐다" 진술 ↔ 조서 상세 설명의 모순

### 4. 기술 작업

- openpyxl: 원본 엑셀에서 별지1 해당 27건 거래일자+금액+거래종류 3중 매칭 후 노란색 형광 처리 (27/27 100%)
- docx-js: ImageRun으로 자금흐름도.png 삽입, 대조표 2개 포함
- cairosvg: SVG→PNG 변환 (2200×1640px)
